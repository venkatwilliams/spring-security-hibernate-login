# Registration and Login Application with Spring Security, Spring Boot, Spring Data JPA, HSQL, JSP, Angular

## Prerequisites
- JDK 1.7 or later
- Maven 3 or later

## Stack
- Spring Security
- Spring Boot
- Spring Data JPA
- Maven
- JSP- (Planning to change UI to Angular JS)
- HSQL

## Run
```mvn clean spring-boot:run```
